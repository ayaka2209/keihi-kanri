# grasp-user-flow — 業務フローの正典

利用者は **青色申告をする個人事業主（単一ユーザー・ローカル運用）**。
目的は「手書きの経費メモを卒業し、確定申告時に年間集計とCSVをボタン一発で出す」こと。

## 業務フロー
1. **経費を入力**: 日付・勘定科目・金額を入れて登録。連続入力しやすいよう、登録後も
   日付と科目は残す（金額だけ変えて続けられる）。
2. **一覧で確認・修正**: 年・月・科目・キーワードで絞り込み、各行を編集/削除。
3. **集計・確定申告**: 年を選ぶと「年間合計」「科目別集計」「月別推移」「CSVダウンロード」。
   - 科目別集計は**確定申告書の経費欄にそのまま転記**できることが価値の中心。
   - CSVは Excel/会計ソフト取り込み用（BOM付きUTF-8で文字化けしない）。

## 勘定科目（青色申告決算書の標準）
初期登録済み: 租税公課 / 荷造運賃 / 水道光熱費 / 旅費交通費 / 通信費 / 広告宣伝費 /
接待交際費 / 損害保険料 / 修繕費 / 消耗品費 / 減価償却費 / 福利厚生費 / 外注工賃 /
利子割引料 / 地代家賃 / 新聞図書費 / 会議費 / 支払手数料 / 雑費。
利用者が独自科目を追加できる（`/api/categories` POST）。
支払方法: 現金 / クレジットカード / 口座振替 / 銀行振込 / 電子マネー / その他。

## 仕様判断の指針
- 金額は税込・整数（円）。マイナスは弾く（`amount >= 0`）。
- 「確定申告でそのまま使えるか」を機能判断の基準にする。会計用語・科目名は勝手に変えない。
- 将来拡張（未着手）: レシートOCR・収入/損益・freee/弥生エクスポート。
  これらに踏み込むときは必ずユーザーに確認。
